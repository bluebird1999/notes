-----------------------------------------------------Hardware Knowledge-----------------------------------------------

* SD type
	SD comes with three size: 
		Standard SD
		Mini SD
		Micro SD
	
	Standard SD comes with three format:
		SD:	  up to 2G
		SDHC: SD High Capacity, up to 32G
		SDSC: SD Extended Capacity, up to 2T

	SD card speed is grouped as:
		Class 2: 2Mbps
		Class 4: 4Mbps
		Class 6: 6Mbps
		Class 10:10Mbps

	Useful refference: https://www.sdcard.org/consumers/cards/
